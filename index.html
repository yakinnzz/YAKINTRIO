💡 POLIMORFISME & ABSTRAKSI

Polimorfisme (banyak bentuk) - objek dapat diperlakukan sebagai bentuk yang berbeda, method yang sama dapat memiliki perilaku berbeda.

Abstraksi - menyembunyikan detail implementasi, hanya menampilkan fungsi penting kepada pengguna.

1. Class Hewan Polimorfisme

Kelas abstrak Hewan memiliki metode bersuara() yang belum diimplementasikan. Subkelas seperti Kucing, Anjing, dan Burung kemudian menyediakan implementasi unik untuk bersuara().

📁 hewan.dart
abstract class Hewan {
  String nama = "Hewan";
  void bersuara();
}

class Kucing extends Hewan {
  Kucing() { nama = "Kucing"; }
  @override void bersuara() => print('$nama: Meong');
}

class Anjing extends Hewan {
  Anjing() { nama = "Anjing"; }
  @override void bersuara() => print('$nama: Guk guk');
}

class Burung extends Hewan {
  Burung() { nama = "Burung"; }
  @override void bersuara() => print('$nama: Cuit cuit');
}

void main() {
  List daftarHewan = [Kucing(), Anjing(), Burung()];
  for (var h in daftarHewan) h.bersuara();
}

2. Class Bentuk Override

Contoh polimorfisme melalui method overriding. Kelas Bentuk memiliki metode gambar() default. Lingkaran dan Persegi menimpa metode ini untuk menampilkan pesan spesifik.

📁 bentuk.dart
class Bentuk {
  void gambar() => print('Menggambar bentuk');
}

class Lingkaran extends Bentuk {
  @override void gambar() => print('Menggambar lingkaran');
}

class Persegi extends Bentuk {
  @override void gambar() => print('Menggambar persegi');
}

void main() {
  List daftarBentuk = [Lingkaran(), Persegi()];
  for (var bentuk in daftarBentuk) bentuk.gambar();
}

3. Operator is & as Type Check

Operator is untuk memeriksa tipe objek pada runtime, operator as untuk melakukan type casting.

📁 kendaraan.dart
class Kendaraan {}
class Mobil extends Kendaraan {
  void klakson() => print('Tin!');
}

void main() {
  Kendaraan k = Mobil();
  if (k is Mobil) {
    print("Cek tipe: k adalah Mobil");
    k.klakson();
  }
  (k as Mobil?)?.klakson();
}

4. Class AlatMusik Abstract Class

Kelas abstrak AlatMusik dengan metode mainkan() yang abstrak (harus diimplementasikan) dan metode info() yang konkret.

📁 alat_musik.dart
abstract class AlatMusik {
  String nama;
  AlatMusik(this.nama);
  void mainkan();
  void info() => print('Ini adalah alat musik $nama');
}

class Gitar extends AlatMusik {
  Gitar(String nama) : super(nama);
  @override void mainkan() => print('Gitar $nama: Jreng jreng...');
}

void main() {
  Gitar gitar = Gitar("Akustik");
  gitar.info();
  gitar.mainkan();
}

5. Interface implements

Konsep interface dalam Dart. Kelas mengimplementasikan abstract class yang bertindak sebagai interface.

📁 interface.dart
abstract class Kendaraan { void jalan(); }
abstract class Terbang { void terbang(); }

  @override void jalan() => print('Pesawat berjalan di landasan');
  @override void terbang() => print('Pesawat terbang di udara');
}

void main() {
  Pesawat pesawat = Pesawat();
  pesawat.jalan();
  pesawat.terbang();
}

6. Static Member Tanpa Instansiasi

Static member dapat diakses langsung melalui nama kelas tanpa membuat objek.

📁 math_utils.dart
class MathUtils {
  static double pi = 3.14;
  static int add(int a, int b) => a + b;
}

void main() {
  print('Nilai pi : ${MathUtils.pi}');
  print('Result   : ${MathUtils.add(10, 20)}');
}

8. Bentuk Geometri Abstraksi & Polimorfisme

Kelas Bentuk mendefinisikan metode hitungLuas() dan hitungKeliling(). Persegi dan Lingkaran mengimplementasikan rumusnya masing-masing.

📁 geometri.dart
abstract class Bentuk {
  double hitungLuas();
  double hitungKeliling();
  void info() => print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}');
}

class Persegi extends Bentuk {
  double sisi;
  Persegi(this.sisi);
  @override double hitungLuas() => sisi * sisi;
  @override double hitungKeliling() => 4 * sisi;
}

class Lingkaran extends Bentuk {
  double radius;
  Lingkaran(this.radius);
  @override double hitungLuas() => 3.14 * radius * radius;
  @override double hitungKeliling() => 2 * 3.14 * radius;
}

void main() {
  Persegi persegi = Persegi(4);
  Lingkaran lingkaran = Lingkaran(7);
  persegi.info();
  lingkaran.info();
}

13. Bonus 5 - Utility Math

Kumpulan fungsi matematika statis seperti faktorial, prima, rata-rata, dan median.

📁 math_utils_full.dart
import 'dart:math';

class MathUtils {
  static int faktorial(int n) => n <= 1 ? 1 : n * faktorial(n-1);
  static bool cekPrima(int n) {
    if(n <= 1) return false;
    for(int i = 2; i <= sqrt(n); i++) if(n % i == 0) return false;
    return true;
  }
  static double rataRata(List list) => list.reduce((a,b)=>a+b) / list.length;
}

void main() {
  print("Faktorial 5 = ${MathUtils.faktorial(5)}");
  print("Prima 7 = ${MathUtils.cekPrima(7)}");
  print("Rata-rata = ${MathUtils.rataRata([10,20,30,40,50])}");
}

14. Challenge - Media Player Polymorphism

Implementasi polimorfisme pada berbagai jenis pemutar media (Audio, Video, Streaming).

📁 media_player.dart
abstract class MediaPlayer { void play(); }

class AudioPlayer extends MediaPlayer {
  @override void play() => print('Memutar Audio MP3...');
}

class VideoPlayer extends MediaPlayer {
  @override void play() => print('Memutar Video MP4...');
}

class StreamingPlayer extends MediaPlayer {
  @override void play() => print('Memutar Live Stream...');
}

class Playlist {
  List media = [];
  void playAll() => media.forEach((m) => m.play());
}

void main() {
  var list = Playlist();
  list.media = [AudioPlayer(), VideoPlayer(), StreamingPlayer()];
  list.playAll();
}
